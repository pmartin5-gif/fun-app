# SVA Workshop: Vibecoding Interactive Web Prototypes

## Context

This is a workshop at SVA (School of Visual Arts) teaching attendees from design and art backgrounds how to vibecode interactive web prototypes. Students are not expected to have prior coding experience.

## Guidelines

- Always give educational context and explanations for technical actions in plain, accessible language. Avoid jargon—if a technical term is necessary, briefly explain what it means.
- When helping with GitHub authentication, always recommend SSH key setup instead of the GitHub CLI.
- Students are using **Claude Code Desktop** and **GitHub Desktop** as their primary tools.
